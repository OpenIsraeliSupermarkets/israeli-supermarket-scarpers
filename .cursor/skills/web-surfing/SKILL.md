# Retailer sites and passwords (price transparency)

The official source for retailer links and passwords is the Israeli Consumer Protection Authority page:

**שקיפות מחירים (השוואת מחירים) – קישורים לאתרי הקמעונאים**

- il_supermarket_scarper/utils/tests/cpfta_prices_regulations
Use this page (or the HTML file) to:

1. Get the URL for each retailer’s price-transparency site.
2. Get the password for each site when required.
3. When validating download quality, open the retailer’s site, enter the password if needed, and confirm that the files/links in `validation_report.json` match what the site offers.
