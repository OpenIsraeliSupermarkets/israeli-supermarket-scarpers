# Retailer sites and passwords (price transparency)

The official source for retailer links and passwords is the Israeli Consumer Protection Authority page:

**שקיפות מחירים (השוואת מחירים) – קישורים לאתרי הקמעונאים**


1. Go to `ScraperFactory` the match the hebrew name to the english one (look at the comment)
2. go to il_supermarket_scarper/utils/tests/cpfta_prices_regulations with the hebrew name and get:
    1. Get the URL for each retailer’s price-transparency site.
    2. Get the password for each site when required.
3. If the site contain more then one chain, make sure to filter it.
4. Filter all todays data points.
5. list all the name of files avaliable to download in the site
